Laser Tilt Motor Test

Running Test

This should be run on partially assembled motors only

All motors must run through EE motor test before this test. Motors must have coupler and encoder installed before this test.

Connecting Motor

You must be grounded for the this step.

Connect the green EtherCAT cable from the qualification bench to the MCB. Make sure the red and black power cable from the MCB to the power board is connected (this should always be there).

Testing the Motor

This test will run the motor through a hysteresis test, moving at low speed one revolution in each direction.

Press 'Continue' to proceed.
