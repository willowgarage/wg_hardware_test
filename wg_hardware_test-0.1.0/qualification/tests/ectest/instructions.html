EtherCAT Diagnostic Test

Setup

Plug in power and EtherCAT cables.

Running

Use power GUI to turn on and off power. Look for green LED on MCB's to check if they have power.

EtherCAT packets will constantly be sent from computers, check for flashing orange lights to check if device is getting packets.

Finishing

Press "Cancel" when you are done with the test to shut down and close the window.
