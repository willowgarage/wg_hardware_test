Forearm Roll Test/Slip Ring Test

Running Test

This test should be run on the PR2 "elbow" assembly. A special cable needs to be connected to its slip ring to allow loop back testing.

Connecting Motor

You must be grounded for the this step.

Connect the green EtherCAT cable from the qualification bench to the MCB. Make sure the red and black power cable from the MCB to the power board is connected (this should always be there).

The calibration flag must be connected to the MCB on the bench in order for this test to work.

Press 'Continue' to proceed.
