Gripper Test

Running Test

Post Burn-in Grippers Only

This test should only be run on grippers after they have completed burn in. If the gripper has sensors, click "Cancel" and select "Gripper Test" from the menu. All new grippers must be qualified with the "Gripper Test", with tips.

Connecting the Gripper

Make sure the gripper is fully secured in the gripper test fixture. Plug the cable from the test fixture into the small plug on the gripper board, through the back of the gripper.

Now hook up the power and ethercat cables from the qualification system into the test fixture.

Check for Debris

Check for any debris in the gears of the gripper. If the gripper has any debris or damage, reject it and report.

Installing Fingertips

This gripper should NOT have fingertips installed. It will fail if fingertips are installed.

Testing the Gripper

Do not touch the gripper unless prompted

Do not interfere with the gripper when it is calibrating, as it may cause the gripper to set its zero position early. Once the gripper fully closes, move it around and watch the visualizer.

The only time you should touch the gripper is while the visualizer is open, and after the gripper is closed. Touching the gripper may cause it to fail its test.

Press 'Continue' to proceed.
