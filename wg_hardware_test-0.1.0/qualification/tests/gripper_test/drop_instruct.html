Gripper Drop Test

Running Test

Run "Gripper Test" First

This test should only be run on a gripper that has been qualified and passed. If a gripper has been rework, modified, or the status is unknown, click "Cancel" and select "Gripper Test" or "Gripper Test - No Tips" from the menu.

Connecting the Gripper

If you have just finished the "Gripper Test" and haven't unplugged the gripper, you are already connected. If not, follow the instructions below.

Make sure the gripper is fully secured in the gripper test fixture. Plug the cable from the test fixture into the small plug on the gripper board, through the back of the gripper.

Now hook up the power and ethercat cables from the qualification system into the test fixture.

Dropping the Gripper

Drop the gripper according to the instructions when prompted. Click "Continue" after every drop. The computer will fail the test automatically if it loses packets from the gripper, or the gripper loses power.

After dropping the gripper, you'll be prompted to visually inspect it. If you see any scratches, dents or marks, press "Cancel" to fail the gripper.

Press 'Continue' to proceed.
