IMU Test

Test Setup

Plug the IMU's USB cable into your computer. Make sure that IMU is sitting flat against the work table.

Do not touch the IMU during the test.



Running IMU Test

The computer will run the test and contact the IMU. It will take approximately 1 minute. During this time, the IMU will calibrate. If you move it during calibration, it will cause the test to fail.

Press 'Continue' to proceed.
